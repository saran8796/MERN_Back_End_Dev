###### **useReducer**



* It is a react hook used to manage complex state logic, when you maintain the two or more states with your component you can use the reducer.
* When the states depend on previous values or has multiple actions.



syntax:



&nbsp;	const \[state, dispatch] = useReducer(reducer, initstate)



* state -> init -> dispatch - trigger action -> value update - init -> Updatedstate .



###### **useRedex**

* One place to keep all the state instead of props.
