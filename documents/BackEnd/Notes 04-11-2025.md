###### **FS (File system):** 



* the FS modules allows you to work with files on your system, to read, write, update, delete , rename, create folders, etc..
* The fs module in nodejs is used to handle file operations like reading, write, creating and deleting the files.
* It has both Blocking, and Non Blocking I/O.



###### **BuildIn functions of FS:**



&nbsp;	- readFile() -> asynchronous

 	- writeFile() -> synchronous

 	- appendFile()

 	- rename()

 	- unlink()

 	- existsSync()
