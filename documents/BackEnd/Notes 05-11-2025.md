###### **Path:**



* The path module provides utils to work with file and directory paths in a way that works on all operating systems.



* To handle file paths cross platform safely, to join resolve and normalize folders and file name, used heavily in file handling and server routing.





###### **URL:**



* URL Module accessing the web address to split or take the part of the url.
* URL module allows you to parse, construct, resolve and manipulate url easily in nodejs, it use when we build an API's or HTTP servers to read query strings or routes.



###### **QueryString:**



* A query string is the extra data added to the end of a URL after the ? to pass information to the server.
* ex: https://example.com/search?keyword=shoes\&sort=price



###### **HTTP:**



* HyperText Transfer Protocol, it allows us to create a webserver that can handle request and response.





###### **Status Code:**



* important codes: 200 - 203, 400-404, 500-504, 511.
