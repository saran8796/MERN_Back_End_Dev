##### **Interview Preparation:**



1. ###### **HTML Essentials**



Focus on:



Semantic tags (header, section, article, footer)



Forms and attributes (required, pattern, placeholder)



Difference between div and span



**What is the DOM?**



**Mini Task:**

Create a small HTML structure for a “Portfolio” page with header, main, footer.



Quick Review Topics:



SEO tags (meta, title, alt)



Accessibility basics (aria-label, alt text)



----------------------------------------------------------------------------



###### 2\. **CSS Mastery**



**Focus on:**



Box model (margin, padding, border)



Flexbox \& Grid (you must be confident with flexbox)



position values (relative, absolute, fixed, sticky)



Responsive design: media queries



Pseudo-classes (:hover, :focus) and animations



**Mini Task:**

Style your HTML portfolio using flexbox for layout and media queries for responsiveness.



----------------------------------------------------------------------------



###### 3\. **JavaScript (Core Logic)**



**Focus on:**



var, let, const – difference



Functions (normal, arrow)



DOM manipulation: document.querySelector, addEventListener



Arrays \& Objects basics



Loops and conditionals



Event bubbling vs capturing



Difference between == and ===



**Mini Task:**

Create a button → when clicked, change background color or show/hide an element.



Pro Tip: They might give a small logic question like reversing a string or finding duplicates in an array.



----------------------------------------------------------------------------



###### 4\. **React.js Basics (If They Use It)**



Focus on:



What is a component?



Functional vs class components



Props and state



useState and useEffect (simple understanding)



JSX rules



Folder structure of a React project



Event handling in React



Mini Task:

Create a simple counter using React (increment/decrement buttons).



----------------------------------------------------------------------------



###### 5\. **Practical + Git + Browser Debugging**



Focus on:



How to inspect elements \& fix CSS/JS errors



console.log() debugging



Basics of Git:



git add, git commit, git push, git pull



What is a repository, branch, merge



----------------------------------------------------------------------------



###### 6\. **Interview Readiness (HR + Tech)**



Prepare Short Answers For:



* “Tell me about yourself.”

&nbsp;	→ Keep it crisp and technical.

&nbsp;	Example:



* “I’m a BCA graduate passionate about front-end development. I’ve built small projects using HTML, CSS, JS, and React. I enjoy building responsive, user-friendly interfaces and I’m currently improving my JavaScript and React skills.”



* “What projects have you built?”

&nbsp;	→ Mention any portfolio, e-commerce, or todo project — even small ones.



* “Why front-end?”



* “Because I enjoy creating visual interfaces and solving UI/UX challenges that improve user experience.”



* “How do you learn new technologies?”



* “I follow documentation, build small practice projects, and use platforms like MDN, W3Schools, and YouTube for quick learning.”



* “Where do you see yourself in 2 years?”



* “I want to become a strong front-end developer proficient in React, TypeScript, and clean UI design.”
